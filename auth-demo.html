🔐 DevNotch Authentication Demo

How Sign Up Works:

1. Enter your name, email, and password

2. System checks if email already exists

3. If new, saves to localStorage as devnotch_users

4. Automatically creates authentication token

5. Stores user profile for quick access

How Login Works:

1. Enter registered email and password

2. System searches localStorage for matching user

3. Validates email and password match

4. Creates authentication token and saves to cookie

5. Saves user profile to localStorage

6. Redirects to dashboard (in real app)

Registered Users (localStorage)

LocalStorage Structure:

Key: devnotch_users

Value: Array of user objects with name, email, password, createdAt

Location: Browser localStorage (persists even after closing browser)
